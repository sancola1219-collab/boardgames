# 3D 棋類遊戲 — 第一人稱對弈

純 **HTML + CSS + JavaScript + SVG** 打造的 3D 棋類遊戲合集，**零依賴、零建置、零框架**。
以第一人稱視角坐在棋桌前對弈，可拖曳環顧、縮放視角。

三款遊戲：

| 遊戲 | 規則重點 | AI |
|---|---|---|
| [象棋](xiangqi/) | 完整規則：將軍／將死／困斃／飛將／重複局面判和 | 迭代加深 α-β 搜尋 + 吃子靜態延伸 + 子力位置評估 |
| [跳棋](checkers/) | 中國跳棋六角星盤，單步與連續跳，先佔滿對面尖角勝（含反堵規則） | 前進度貪婪／多層 α-β（依難度） |
| [黑白棋](reversi/) | 標準翻轉棋，雙方無步即終局 | α-β + 局面權重 + 行動力 + 終盤精算 |

## 功能

- **四種模式**：雙人同機對戰、與電腦對戰、電腦對電腦（觀戰，可調速/暫停）、**連線對戰**
- **三段電腦難度**：簡單／中等／困難（搜尋深度、時間預算與隨機性不同）
- **第一人稱 3D 視角**：拖曳環顧、滾輪或雙指縮放、一鍵回復視角
- **對弈輔助**：合法步提示、最後一手標記、悔棋（對電腦悔棋自動退回整回合）
- **連線對戰免伺服器**：WebRTC DataChannel 點對點直連，複製邀請碼／回應碼即可配對（兩台裝置各開同一頁面）

## 遊玩

直接開啟網頁即可，無需安裝：

- 線上：GitHub Pages（本 repo 的 Pages 網址）
- 離線：`git clone` 後直接雙擊 `index.html`，或起一個靜態伺服器（例如 `python -m http.server`）

操作：拖曳旋轉視角、滾輪／雙指縮放、點選棋子再點目的地落子。

## 技術架構

```
assets/engine3d.js   共用 3D 引擎：透視相機、軌道控制、射線拾取、SVG 繪製（棋石/圓柱/陰影）
assets/shell.js      共用外殼：頂欄、開局對話框（模式/難度/先後手）、連線配對流程、勝負橫幅
assets/net.js        WebRTC 連線模組（手動信令，免伺服器）
assets/style.css     共用深色 UI
xiangqi/  checkers/  reversi/   各遊戲規則、AI 與渲染
```

- 3D 全部用 SVG 手寫透視投影（無 WebGL/Three.js），棋子以畫家演算法按深度排序
- 遊戲狀態與動畫分離：棋步立即生效，動畫僅為視覺重播，背景分頁節流不影響對局
- AI 以時間預算 + 迭代加深控制強度，超時自動退回上一層最佳解
