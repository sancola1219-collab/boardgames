# 3D 遊戲合集

純 **HTML + CSS + JavaScript + SVG** 打造的 3D 網頁遊戲合集，**零依賴、零建置、零框架**。
所有遊戲皆為第一人稱 3D 視角，可拖曳環顧、滾輪／雙指縮放，並支援電腦對戰與難度調整。

**線上遊玩：https://sancola1219-collab.github.io/boardgames/**

## 收錄遊戲（8 款）

### 棋類（支援雙人同機、電腦三段難度、WebRTC 連線對戰）
| 遊戲 | 規則重點 | AI |
|---|---|---|
| [象棋](xiangqi/) | 完整規則：將軍／將死／困斃／飛將／重複局面判和 | 迭代加深 α-β + 吃子靜態延伸 |
| [跳棋](checkers/) | 中國跳棋六角星盤，單步與連跳，先攻頂者勝 | 前進度評估／多層 α-β |
| [黑白棋](reversi/) | 標準翻轉棋，雙方無步即終局 | α-β + 局面權重 + 終盤精算 |

## 共同特色

- **第一人稱 3D 視角**：拖曳環顧、滾輪／雙指縮放、一鍵回復視角
- **電腦對手與難度**：每款可調 AI 強度
- **手機友善**：觸控操作、螢幕方向鍵／動作鍵
- **連線對戰**（棋類）：WebRTC DataChannel 點對點直連，複製邀請碼／回應碼即可配對，免伺服器
- **可離線遊玩**：純前端，`index.html` 直接雙擊即可

## 技術架構

```
assets/engine3d.js   共用 3D 引擎：透視相機、軌道控制、射線拾取、SVG 繪製（棋石/圓柱/方塊/陰影）
assets/cards3d.js    3D 平放卡牌繪製（UNO／撲克共用）
assets/shell.js      共用外殼：頂欄、開局對話框、連線配對、勝負橫幅
assets/net.js        WebRTC 連線模組（手動信令，免伺服器）
assets/style.css     共用深色 UI
xiangqi/ checkers/ reversi/ uno/ poker/ tetris/ snake/ monopoly/   各遊戲規則、AI 與渲染
```

- 3D 全部用 SVG 手寫透視投影（無 WebGL/Three.js），依畫家演算法按深度排序
- 遊戲狀態與動畫分離：邏輯立即生效，動畫僅為視覺重播，背景分頁節流不影響對局
- 隱藏分頁時渲染迴圈自動改用計時器後備，確保背景仍能運作
